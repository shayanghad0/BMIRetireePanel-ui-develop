# پروژه مدیریت امور بازنشستگان بانک ملی ایران (فرانت‌اند)

این یک پنل مدیریت امور اداری بازنشستگان تنظیم شده بر اساس نیاز های بانک ملی ایران هست 
این پروژه توانایی دریافت احکام اداری ، فبش حقوقی ، اطلاعات مستمری و .. برای بازنشستگان را دارد

## Stack

*   **Technology:** React.js، Vite.js
*   **UI Design :** tailwindcss , material ui
*   **Routing:** react-router

## Setup

برای راه‌اندازی و اجرای این پروژه روی سیستم خود، مراحل زیر را دنبال کنید:


```bash
npm clone https://github.com/aryan-zargar/BMIRetireePanel.git
cd BMIRetireePanel
pip install
# یا اگر از Yarn استفاده می‌کنید:
# yarn install
npm run dev
